HTML

HTML (Hypertext Markup Language) is the standard markup language for creating web pages and web applications. With Cascading Style Sheets (CSS) and JavaScript, it forms a triad of cornerstone technologies for the World Wide Web.

HTML Elements

HTML documents are made up of HTML elements, which are defined by tags. Each tag begins with a "<" character and ends with a ">" character, and most tags have a corresponding closing tag that begins with "". Here are some common HTML elements:

  • <p>: defines a paragraph
  • <h1> to <h6>: define headings
  • <a>: defines a hyperlink
  • <img>: defines an image
  • <ul>: defines an unordered list
  • <ol>: defines an ordered list
  • <table>: defines a table
  • <form>: defines a form for user input

HTML Attributes

HTML elements can also have attributes, which provide additional information about the element. Attributes are always specified in the opening tag, and are written as name-value pairs. Here are some common HTML attributes:

  • class: specifies one or more class names for an element
  • id: specifies a unique id for an element
  • src: specifies the URL of an image or other media element
  • href: specifies the URL of a link
  • style: specifies inline CSS styles for an element
  • alt: specifies alternative text for an image
  • title: specifies additional information about an element (displayed as a tooltip)
CSS

CSS (Cascading Style Sheets) is used to style and layout web pages, giving them a consistent look and feel. CSS can be applied to HTML documents using either inline styles, internal styles, or external stylesheets.

Inline Styles

Inline styles are styles that are applied directly to individual HTML elements, using the style attribute. Here is an example:

This paragraph is red and has a font size of 24 pixels.

Inline styles have the highest precedence of all styles, meaning that they will override any other styles applied to the same element.

Internal Styles

Internal styles are styles that are defined within the "head" section of an HTML document, using the "style" tag. Internal styles have a lower precedence than inline styles, but a higher precedence than external stylesheets.

External Stylesheets

External stylesheets are styles that are defined in a separate file, and then linked to from the HTML document using the "link" tag External stylesheets have the lowest precedence of all styles, meaning that they will be overridden by any styles defined in inline styles or internal styles.

JavaScript

JavaScript is a scripting language that is used to add interactivity to web pages. It is primarily used for client-side scripting, meaning that it runs in the user's web browser and can manipulate the HTML content and CSS styles of a web page.

Here is an example of a simple JavaScript program:

function greet() {
alert('Hello, world!');
}
greet();

This program defines a function called greet() that takes no parameters. When the function is called, it will print the string "Hello, world!" to the console.

Here's how you would call the function:

greet();

Control Structures

JavaScript provides several control structures that allow us to execute different blocks of code depending on certain conditions. These include:

  • if/else: executes a block of code if a specified condition is true, and another block of code if it is false.
  • switch: executes different blocks of code depending on the value of an expression.
  • for: executes a block of code a specified number of times.
  • while: executes a block of code while a specified condition is true.

Here is an example of an if/else statement:

var x = 5;
if (x > 10) {
  console.log("x is greater than 10");
} else {
  console.log("x is less than or equal to 10");
}

In this example, we define a variable called x and assign it the value of 5. We then use an if/else statement to check if x is greater than 10. Since x is not greater than 10, the code inside the else block is executed, and the output is:

x is less than or equal to 10

Here is an example of a for loop:

for (var i = 0; i < 5; i++) {
  console.log(i);
}

This code defines a for loop that will execute the block of code inside it five times. The loop initializes a variable called i to 0, checks if i is less than 5, executes the block of code inside the loop, and then increments i by 1. This process repeats until i is no longer less than 5. The output of this code is:

0
1
2
3
4

Python

Python is a high-level, interpreted programming language that is widely used for web development, scientific computing, data analysis, artificial intelligence, and more. It was first released in 1991 by Guido van Rossum, and its design philosophy emphasizes code readability and ease of use, making it a popular choice for beginners and experienced programmers alike.

Python Variables

In Python, variables are used to store values that can be accessed and manipulated throughout the program. Unlike some other programming languages, Python does not require you to declare the type of a variable before you use it.

Here's an example of how to define a variable in Python:

x = 5
y = "Hello, world!"

In this example, we define two variables: x and y. x is assigned the value 5, and y is assigned the string "Hello, world!".

Python Data Types

Python supports a variety of data types, including numbers, strings, booleans, lists, tuples, and dictionaries. Unlike some other programming languages, Python is able to automatically determine the data type of a variable based on the value assigned to it.

Here are some examples of different data types in Python:

num = 42             # integer
pi = 3.14            # float
name = "John"        # string
is_student = True    # boolean
fruits = ["apple", "banana", "cherry"]      # list
coordinates = (10, 20)                      # tuple
person = {"name": "John", "age": 25}        # dictionary

As you can see, Python makes it easy to work with different types of data.

Loops

In Python, there are two main types of loops: for loops and while loops.

A for loop is used to iterate over a sequence (such as a list or a string) a specific number of times:

fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
print(fruit)

In this example, the for loop iterates over the fruits list, assigning each element to the variable fruit in turn. The print statement then prints each element.

A while loop is used to repeat a block of code as long as a certain condition is true:

i = 1
while i < 6:
print(i)
i += 1

In this example, the while loop prints the values of i from 1 to 5 (inclusive), incrementing i by 1 each time through the loop.

Python Functions

Functions are an essential part of any programming language, and Python is no exception. A function is a block of code that performs a specific task, and can be called from other parts of the program as needed.

Here's an example of a Python function:

def greet():
print("Hello, world!")

This program defines a function called greet() that takes no parameters. When the function is called, it will print the string "Hello, world!" to the console.

Here's how you would call the function:

greet()

Conclusion

Python is a powerful and versatile programming language that can be used for a wide variety of tasks, from simple scripts to complex web applications. Its simple syntax and ease of use make it a popular choice for beginners, while its powerful libraries and frameworks make it a favorite among experienced developers.

Ruby

Ruby is a dynamic, object-oriented programming language that was designed to balance programmer productivity and simplicity with performance and scalability. It has a clean and concise syntax that makes it easy to read and write, and it is often used for web development, database programming, and other tasks that require a high level of abstraction. Ruby was first released in 1995 by Yukihiro "Matz" Matsumoto, and it is designed to be easy to read and write, making it a popular choice for beginners and experienced programmers alike.

Variables and Data Types

In Ruby, variables are declared using the = operator. Ruby is a dynamically-typed language, which means that the type of a variable is determined at runtime based on the value assigned to it.

Here is an example of a variable declaration in Ruby:

x = 5
y = "Hello, world!"

In this example, x is assigned the integer value 5, and y is assigned the string value "Hello, world!".

Ruby has several built-in data types, including integers, floats, strings, symbols, arrays, and hashes.

Conditional Statements

Conditional statements in Ruby use the if, elsif, and else keywords. Here is an example of an if statement in Ruby:

age= 18
if age >= 18
puts "You are old enough to vote."
end

This program defines a variable called age and checks if it is greater than or equal to 18. If it is, the program prints "You are old enough to vote." to the console. If not, nothing happens.

Loops

In Ruby, there are two types of loops; for loops and while loops, but they are not commonly used. Instead, the each method is often used to iterate over arrays and other collections. Here is an example:

fruits = ["apple", "banana", "cherry"]
fruits.each do |fruit|
  puts fruit
end

This program defines an array called fruits and uses the each method to iterate over each element in the array and print it to the console.
